== Description ==

ZubairRay Quick CTA Bar is a lightweight and easy-to-use WordPress CTA bar plugin that helps website owners increase engagement, generate leads, and drive more conversions with a sticky call to action bar.

Display a responsive notification bar at the top or bottom of your website with a custom message and action button. Encourage visitors to call, send a WhatsApp message, email you, or visit any custom URL.

Whether you run a business website, local service company, agency, online store, blog, or portfolio, ZubairRay Quick CTA Bar makes it easy to highlight important offers, contact options, promotions, and announcements.

= Why Choose ZubairRay Quick CTA Bar? =

Many websites struggle to get visitors to take action. A clear and visible call to action bar can significantly improve user engagement by keeping important contact options and offers visible at all times.

Use the plugin to increase phone calls, generate WhatsApp inquiries, collect email leads, promote special offers, highlight announcements, and improve customer engagement.

= Lightweight & Privacy Friendly =

ZubairRay Quick CTA Bar is designed with performance in mind. The plugin does not require external services, does not collect personal data, and keeps your website fast and lightweight.



== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/zubairray-quick-cta-bar` directory, or install the plugin through the WordPress plugins screen.
2. Activate the plugin through the Plugins screen in WordPress.
3. Go to Quick CTA Bar in the WordPress admin menu.
4. Configure your message, CTA type, destination, design, and visibility settings.

== Frequently Asked Questions ==

= Does this plugin collect visitor data? =

No. The plugin does not collect personal data or send data to external services.

= Does the remember closed state use cookies? =

Yes. If enabled, the plugin stores a simple local cookie so the closed bar stays hidden for returning visitors.

= Can I show the bar only on mobile or desktop? =

Yes. The plugin includes desktop, mobile, mobile-only, and desktop-only visibility controls.

= Does it work with block themes and classic themes? =

Yes. The bar is rendered on the frontend footer and works with Gutenberg, Elementor, WooCommerce, classic themes, and block themes.
